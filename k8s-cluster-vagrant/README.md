# k8s-cluster-vagrant

* Vagrant only k8s cluster setup
* A sweet, neat and simple solution
* gets you ready with k8s cluster in 20 mins

#### How to Setup?

* run `vagrant up`
* run `vagrant ssh k8s-master`
* run `kubectl get nodes`
* allow some time to get below result (expected)
